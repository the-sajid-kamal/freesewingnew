## Construction {#construction}
### Elastic width {#elasticwidth}

**The width of the elastic band you use for the underbust band**
- Type: **Percentage**
- Default: **2%**
- Minimum: **0.5%**
- Maximum: **5%**

<Elasticwidth />

## Fit {#fit}
### Armpit adjustment {#armpitadjustment}

**Move the armpit up or down**
- Type: **Percentage**
- Default: **-1.5%**
- Minimum: **-10%**
- Maximum: **10%**

<Armpitadjustment />

### Bust point focus {#bustpointfocus}

**How much the virtual bust darts aim at the bust point**
- Type: **Percentage**
- Default: **0%**
- Minimum: **-50%**
- Maximum: **100%**

<Bustpointfocus />

### Elastic band stretch {#horizontalbandstretch}

**How much the fabric will stretch along the elastic band, when worn**
- Type: **Percentage**
- Default: **15%**
- Minimum: **0%**
- Maximum: **30%**

<Horizontalbandstretch />

### Bust stretch {#horizontalbuststretch}

**How much the fabric will stretch along the bust line, when worn**
- Type: **Percentage**
- Default: **15%**
- Minimum: **0%**
- Maximum: **30%**

<Horizontalbuststretch />

### Length {#length}

**The length of the garment**
- Type: **Percentage**
- Default: **22.5%**
- Minimum: **0%**
- Maximum: **100%**

<Length />

### Strap angle {#strapangle}

**The angle of the straps on your shoulder line**
- Type: **Degrees**
- Default: **25°**
- Minimum: **0°**
- Maximum: **45°**

<Strapangle />

### Strap position {#strapposition}

**Move the straps towards your shoulders or neck**
- Type: **Percentage**
- Default: **40%**
- Minimum: **20%**
- Maximum: **60%**

<Strapposition />

### Strap stretch {#strapstretch}

**How much the fabric will stretch along the straps, when worn**
- Type: **Percentage**
- Default: **15%**
- Minimum: **0%**
- Maximum: **50%**

<Strapstretch />

### Vertical stretch {#verticalstretch}

**How much the fabric will stretch vertically, when worn**
- Type: **Percentage**
- Default: **11%**
- Minimum: **0%**
- Maximum: **30%**

<Verticalstretch />

## Style {#style}
### Back width {#backwidth}

**The width of narrow part on the back**
- Type: **Percentage**
- Default: **18%**
- Minimum: **5%**
- Maximum: **40%**

<Backwidth />

### Neck depth back {#neckheightback}

**The depth of the neck hole at the back**
- Type: **Percentage**
- Default: **17%**
- Minimum: **10%**
- Maximum: **25%**

<Neckheightback />

### Neck depth front {#neckheightfront}

**The depth of the neck hole at the front**
- Type: **Percentage**
- Default: **35%**
- Minimum: **25%**
- Maximum: **80%**

<Neckheightfront />

### Strap width {#strapwidth}

**The width of the shoulder straps**
- Type: **Percentage**
- Default: **33%**
- Minimum: **15%**
- Maximum: **50%**

<Strapwidth />

## Advanced {#advanced}
### Back dart angle {#backdartangle}

**The angle of the virtual back dart**
- Type: **Degrees**
- Default: **15°**
- Minimum: **0°**
- Maximum: **22°**

<Backdartangle />

### Lower back shape {#lowerbackshape}

**The lower curve of the C shape on the back**
- Type: **Percentage**
- Default: **40%**
- Minimum: **20%**
- Maximum: **80%**

<Lowerbackshape />

### Back split offset {#splitoffsetback}

**Shifts the back seam towards the side or the back**
- Type: **Percentage**
- Default: **28%**
- Minimum: **15%**
- Maximum: **40%**

<Splitoffsetback />

### Front split offset {#splitoffsetfront}

**Shifts the front seam towards the side or the bust**
- Type: **Percentage**
- Default: **30%**
- Minimum: **15%**
- Maximum: **40%**

<Splitoffsetfront />

### Strap curve front {#strapcurvefront}

**How straight the shoulder straps are on the front**
- Type: **Percentage**
- Default: **5%**
- Minimum: **2%**
- Maximum: **10%**

<Strapcurvefront />

### Upper back shape {#upperbackshape}

**The upper curve of the C shape on the back**
- Type: **Percentage**
- Default: **40%**
- Minimum: **20%**
- Maximum: **60%**

<Upperbackshape />
